Atividades Práticas - Desenvolvimento Web 1
Este repositório contém uma coleção de projetos e atividades práticas desenvolvidas para a disciplina de Desenvolvimento Web 1. O objetivo é demonstrar o aprendizado e a aplicação de conceitos de HTML CSS e JS através de diversas páginas web interativas.

Visão Geral
A página principal simula uma interface de sistema operacional, apresentando uma "janela" com links para diferentes projetos. Há também uma barra de tarefas com ícones decorativos e uma imagem de perfil.

Tecnologias Utilizadas
HTML5: Para estruturação do conteúdo da página.
CSS3: Para estilização e layout, incluindo responsividade básica.
JavaScript: Utilizado nos projetos individuais para adicionar interatividade, lógica de programação e manipulação dinâmica de elementos.
